# Изображения

Эта папка содержит изображения для сайта.

## Необходимые файлы:

1. **logoPivovar.png** - Логотип компании для хедера
   - Размер: рекомендуется 80x80px или больше (квадратное изображение)
   - Формат: PNG с прозрачностью
   - Расположение: `/public/images/logoPivovar.png`

2. **aboutPivovar.png** - Изображение для секции "О ДомПивоваре"
   - Размер: рекомендуется квадратное изображение (например, 600x600px)
   - Формат: PNG
   - Расположение: `/public/images/aboutPivovar.png`

## Дополнительные изображения для статей:

Для корректной работы секции "Статьи" в сайдбаре, добавьте следующие изображения:

- `/public/images/articles/yarfest-2023.jpg`
- `/public/images/articles/beer-brewing-process.jpg`
- `/public/images/articles/vienna-beer.jpg`

Эти изображения можно заменить на placeholder или удалить соответствующие статьи из компонента `SidebarSections`.
