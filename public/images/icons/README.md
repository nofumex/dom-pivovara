# Иконки

Эта папка содержит иконки для сайта.

## Необходимые файлы:

1. **gift-icon.png** - Иконка подарочного сертификата
   - Размер: рекомендуется 32x32px или больше
   - Формат: PNG с прозрачностью
   - Расположение: `/public/images/icons/gift-icon.png`
   - Использование: Секция преимуществ - "Подарочные сертификаты"

2. **check-icon.png** - Иконка галочки/сертификата
   - Размер: рекомендуется 32x32px или больше
   - Формат: PNG с прозрачностью
   - Расположение: `/public/images/icons/check-icon.png`
   - Использование: Секция преимуществ - "Весь товар сертифицирован"

3. **refresh-icon.png** - Иконка обмена/возврата
   - Размер: рекомендуется 32x32px или больше
   - Формат: PNG с прозрачностью
   - Расположение: `/public/images/icons/refresh-icon.png`
   - Использование: Секция преимуществ - "30 дней на обмен и возврат"

4. **truck-icon.png** - Иконка доставки
   - Размер: рекомендуется 32x32px или больше
   - Формат: PNG с прозрачностью
   - Расположение: `/public/images/icons/truck-icon.png`
   - Использование: Секция преимуществ - "Удобная и быстрая доставка"

5. **search-icon.png** - Иконка поиска
   - Размер: рекомендуется 20x20px или больше
   - Формат: PNG с прозрачностью
   - Расположение: `/public/images/icons/search-icon.png`
   - Использование: Кнопка поиска в хедере

## Рекомендации:

- Все иконки должны иметь прозрачный фон
- Цвет иконок должен быть оранжевым (#F57C00 или var(--color-primary-600))
- Иконки должны быть четкими и хорошо читаемыми на белом фоне
- Для лучшего качества используйте формат PNG с прозрачностью
